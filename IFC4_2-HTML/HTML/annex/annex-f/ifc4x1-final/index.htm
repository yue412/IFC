F.7 IFC4x1 Final

The final release of IFC4.1 incorporates all release candidate changes, documentation corrections, and includes the following schema changes:

The entity IfcClothoidalArcSegment2D has been deleted and replaced with IfcTransitionCurveSegment2D.

The entity IfcTransitionCurveSegment2D has StartRadius and EndRadius changed to use positive lengths.
