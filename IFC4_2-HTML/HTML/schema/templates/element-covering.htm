4.7.5 Element Covering

Covering elements are assigned to their hosting elements using the Element Covering concept.

EXAMPLE  A cladding is assigned to a wall, a flooring is assigned to a slab, a wrapping is assigned to a duct using the IfcRelCoversBldgElements relationship.
